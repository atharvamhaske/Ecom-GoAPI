## Features

<AccordionGroup>
  <Accordion icon="box" title="Product Management">

  </Accordion>
  <Accordion icon="gauge-high" title="High Performance">

    Designed to scale horizontally with stateless endpoints and efficient database queries.

## Tech Stack

This API is built with modern technologies:

- **Go** - High-performance backend language
- **Chi Router** - Lightweight, idiomatic HTTP router
- **PostgreSQL** - Robust relational database
- **SQLC** - Type-safe SQL query generation
- **Docker** - Containerized deployment

## Quick Example

Here's a simple example of fetching all products:

```bash
curl http://localhost:8080/api/v1/products
```

Response:
```json
[
  {
    "id": 1,
    "name": "Laptop",
    "description": "High-performance laptop with 16GB RAM",
    "price": 999.99,
    "stock": 50,
    "created_at": "2025-01-15T10:30:00Z",
    "updated_at": "2025-01-15T10:30:00Z"
  }
]
```
